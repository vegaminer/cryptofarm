You can disable User Access Control via GUI by following this [[:windows:tuning:disable_uac|guide]]

----

=== Prerequisites ===

1. [[windows:rigutils:install|Install]] Rigutils
2. I highly recommend to complete [[windows:rigutils:windows_tuning:antivirustest.bat|AntiVirusTest.bat]] step before executing //DisableUAC.bat//
3. Start [[windows:rigutils:windows_tuning:cmd_rigutils.bat|cmd_rigutils.bat]]

----
